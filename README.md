# Essence ✨

An AI-powered app that turns your journal entries into visual art based on how you're feeling.

> This project is in early development. Currently figuring out sentiment analysis + image generation.

## What It Does
Essence reads your journal entries, analyzes the emotional tone, and generates visual art that reflects your inner state. The idea is to make self-reflection more intuitive and beautiful.

## Planned Stack
- Python (backend + sentiment analysis)
- Gemini API (AI + image generation)
- HTML/CSS front end

## Status
🚧 Work in progress

Currently exploring:
- Sentiment analysis models
- Image generation via Gemini API
- UI/UX design for emotional clarity

## Vision
Built at the intersection of mental health and creative technology. The goal is an app that doesn't just store your thoughts — it transforms them into something you can see and feel.
